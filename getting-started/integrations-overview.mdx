Syncly integrations bring all of your customer feedback into one place, so your team never has to hunt across a dozen tabs to understand what users are saying. Whether your customers reach you through a support ticket, a post-call survey, a community thread, or an app store review, each integration routes that signal directly into the **Feedback Hub**, where Syncly's AI engine tags, scores, and surfaces the insights that matter most.

## What Integrations Do

When you connect a data source, Syncly establishes a secure, authenticated link to that tool and begins pulling conversation data automatically. New data arrives in near-real time (or on a daily schedule, depending on the source), so your Feedback Hub stays current without any manual effort on your part. Every connected source adds more signal for the AI — the more sources you connect, the clearer the picture you get of your customers' experience.

## All Available Integrations

The table below lists every integration Syncly currently supports, along with its category and what it contributes to the Feedback Hub.

| Integration | Category | Data Synced |
|---|---|---|
| Zendesk | Customer Support | Tickets, tags, CSAT scores, agent notes |
| Intercom | Customer Support | Conversations, chat messages, attributes |
| Front | Customer Support | Shared inbox threads, tags, comments |
| HelpDesk | Customer Support | Tickets, replies, satisfaction ratings |
| Atlas | Customer Support | Support conversations, customer health data |
| Gorgias | Customer Support | E-commerce support tickets, macros, tags |
| Salesforce | CRM | Contacts, accounts, opportunities, notes |
| HubSpot | CRM | Contacts, deals, notes, lifecycle stage |
| SurveyMonkey | Survey | Survey responses, NPS scores, ratings |
| Typeform | Survey | Form responses, NPS, open-ended answers |
| Gmail | Email | Customer email threads and feedback |
| Grain | Call Recording | Call transcripts, highlights, notes |
| Gong | Call Recording | Call recordings, transcripts, AI insights |
| Slack | Community | Customer community messages and threads |
| Discord | Community | Server messages and community discussions |
| Google Play | Online Store | App reviews, star ratings, reply threads |
| Zapier | Automation | Any source via custom Zap pipelines |
| Excel / CSV | Automation | Bulk historical or one-off data uploads |
| Jira | Output | Push Syncly insights as Jira issues |
| Linear | Output | Push Syncly insights as Linear issues |

<Tip>
**Connect at least three sources for the richest insights.** Syncly's AI cross-references signals across channels — for example, correlating a spike in support tickets with a drop in app store ratings — to surface patterns that a single-source view would miss. The more data you bring in, the more confidently the AI can identify trending topics and at-risk segments.
</Tip>

## Manual Imports

If you have historical feedback data in a spreadsheet, or you use a tool that doesn't have a native Syncly integration yet, you can upload it directly. See [Import Feedback Data via CSV or Excel File](/getting-started/import-csv) for step-by-step instructions, column requirements, and file size limits.
